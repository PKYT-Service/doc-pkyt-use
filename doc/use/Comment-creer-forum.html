Comment créer un forum avec SYS PKYT

config.json

Le fichier config.json permet de gérer les redirections en cas d'erreur ou de déconnexion. Remplacez l'email par "false" si vous ne souhaitez pas en fournir. ⚠️ Merci de ne pas modifier les "version_x" et le "version".

{
  "err_login": "error/error_login.html",
  "err_account_invalid": "error/invalid_account.html",
  "deconnecter": "error/logout.html",
  "connecter": "admin.html",
  "votre_lcid": "",
  "moyen_de_contact": "",
  "pseudo_de_contact": "",
  "email": "",
  "service": "votre_service",
  "version_j": "v1.7",
  "version_l": "v1ab",
  "version_a": "v1ac",
  "version_p": "v1ab",
  "version": "2.09"
}
