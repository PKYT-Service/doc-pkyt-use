PKYT - Service // LosSantos Police Departement By EnesGP

Simple pack Loader pour RPH incluant des mods est des PG(=PlugIn) pour LSPDFR.



Installations

1. Installation logique : 

    Pour commencer, veuillez installer le pack LSPDFR, puis extrayez-le dans le dossier racine de GTA 5.
    Le lien : [Google Drive / Safe] https://drive.google.com/file/d/1OsseeqgsDwevtqG_6GWznhhRWUQF8v8c/view?usp=sharing
    Et rage COOP pour le MutliPlayer

Installation vehicules : 

Chaque véhicule est compatible avec ELS.  

Il suffit simplement de les glisser dans le bon répertoire à l'aide d'OpenIV :  

 Dossier des véhicules :
-   Sheriff / Sheriff2 : update/x64/dlcpacks/patchday20ng/dlc.rpf/...
-   FBI / FBI : update/x64/dlcpacks/patchday22ng/dlc.rpf/...

 Dossier OIV :
- _PACK Voiture_ : LSPD Pack Installer 1.5.0.oiv (à installer avec OpenIV).
- _Gun Porter_   : Slaker's Realistic Weapons Mechanics 4.1.oiv (à installer avec OpenIV).
